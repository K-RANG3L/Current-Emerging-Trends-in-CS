# Current-Emerging-Trends-in-CS

CS 370 Portfolio Reflection

Project Overview

For this project, I completed the intelligent agent for a pirate treasure hunt game using reinforcement learning. The starter code included the game environment, maze, and supporting functions. My responsibility was to implement the Deep Q-Learning algorithm so the pirate could learn the best path to the treasure. I trained the model, tested its performance, and evaluated the results to ensure the agent successfully completed the maze.

What I Learned

This project taught me how reinforcement learning allows an agent to learn through experience instead of following hard-coded instructions. I gained experience working with neural networks, reward systems, exploration versus exploitation, and training an AI model to improve its performance over time.

How This Connects to Computer Science

Computer scientists design algorithms and software that solve complex problems efficiently. Artificial intelligence is an important area of computer science because it enables computers to make decisions based on data and experience. The concepts I learned in this course can be applied to robotics, autonomous vehicles, game development, recommendation systems, and many other real-world applications.

My Approach to Problem Solving

I approach problems by first understanding the requirements, then breaking the project into smaller, manageable tasks. Throughout this project, I tested my code often, analyzed the agent's performance, and made improvements until it consistently found the treasure. This process helped me develop stronger debugging and problem-solving skills.

Ethical Responsibilities

As a computer scientist, I have a responsibility to create software that is reliable, secure, and fair. When developing AI systems, it is important to reduce bias, protect user data, and ensure that intelligent systems make decisions responsibly. Ethical development helps build trust between organizations and the people who use their technology.
